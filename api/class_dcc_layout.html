class DccLayout¶

Main interface for creating the Layout mode. Once the command line parameters set and stored in the configuration a call to build() will read the files, validate the layout description and generate the paths applicable for the layout.

Note

Capacty is not yet considered. This will become important for layout control i.e. sending a given train only on routes where the endpoint has enough capacity to hold the train.
